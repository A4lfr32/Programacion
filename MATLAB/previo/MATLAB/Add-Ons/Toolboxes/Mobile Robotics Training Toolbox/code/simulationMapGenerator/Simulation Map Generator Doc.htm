Simulation Map Generator App

Generate simulation maps for line following and obstacle detection applications

 

Description

Import formatted maps/sketches/pictures of courses/fields to generate simulation maps for line following and obstacle detection applications. Use the map with the Mobile Robotics Training library blocks to simulate closed-loop mobile robotics systems.

 

App Window Options

Map Initialization

Load Map Image

Browse to a location and select the image to be loaded.

Note: Upload top view images of the course/field. If the images are not exactly top views, this will cause inaccuracies in distance calculations. The simulator will still work, but the algorithm values may not work in the hardware and may need to be re-tuned.

Track Size [X Y] (m)

Specify actual track size in meters. X is along the horizontal and Y is along the vertical.

Note: This is an important parameter that controls the distance calculations during simulation. Provide an accurate value.

Map Options

Choose the type of map to generate and export. Following options are available:

Line Following – Use this option to create only line following maps.

Obstacle – Use this option to create only obstacle maps.

Both – Use this option to create combined line following and obstacle maps.

Line Following Map

Threshold Mode

Use this option to choose the type of thresholding to generate the line following map. Two options are available as follows:

Automatic

This option uses the graythresh function to automatically threshold the image.

Manual

This option manually uses the level specified by the Threshold Level slider to threshold the image.

Threshold Level

Threshold value to use by the Manual thresholding option.

Note: For formatted images, Automatic option should work well. For camera pictures, choose Manual. Try to use pictures that have even lighting conditions.

Obstacle Map

Number of Obstacles

Represents the number of obstacles in the obstacle map.

Place Obstacles

Opens a new window with the original map image. Click and drag a rectangle around the obstacle area, Double click inside the rectangle to place an obstacle. Repeat this process in the window for as many times as specified by the Number of Obstacles parameter.

Note: Approximate obstacles as rectangular regions.

Map Preview

Automatically updates current map preview based on map generation parameters.
